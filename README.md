# Mirage Engine

**Mirage Engine** is a **VR-only game engine** built with **C# 9.0**, designed specifically for immersive virtual reality experiences. It focuses on performance, low-latency input, and full 6DoF interaction for next-generation VR games.  

> Mirage Engine is purpose-built exclusively for VR, supporting PC and Android VR platforms.

---

## Features

- **VR-Only Architecture** – Built from the ground up for virtual reality  
- **C# 9.0 Powered** – Modern, fast, and developer-friendly  
- **Cross-Platform VR Support** - Windows, Meta Quest, PICO, and other Android VR devices  
- **6DoF Head & Hand Tracking**  
- **Low-Latency Input System**  
- **Physics-Based Interaction**  
- **Custom Rendering Pipeline (VR-Optimized (Scriptable and Universal)**  
- **Modular Engine Design**  
- **High Performance & Memory Efficient**  
- **OpenXR Ready**  

---

## Engine Philosophy

Mirage Engine is designed for:  
- Immersion first  
- No compromises for VR performance  
- Physics-driven gameplay  
- Developer control over performance  
- VR-native workflows  

Unlike traditional engines, Mirage does **not** support non-VR cameras or 2D game pipelines.

---

## Tech Stack (for the nerds out there)

- **Language:** C# 9.0  
- **Architecture:** Modular ECS-style design  
- **Graphics:** VR-optimized rendering pipeline  
- **Input:** Motion controllers + headset tracking  
- **XR Standard:** OpenXR-ready  
- **Platforms:** Windows VR, Meta Quest, PICO, Android VR
